This is a code repository for the project 'SRPSS [Scientific Research Paper Submission System]'. 
I developed this code with the Visual Studio Code, Git, XAMPP Server.
Tech Stack:-
HTML5
CSS3
Vanilla JavaScript
AJAX
PHP 8.0+
MySQL
SQL

Instructions:-
1. First of all, download the code repository.
2. Now, run the Apache server and MySQL if you are using XAMPP/WAMPP. 
3. Now, open the phpmyadmin in your web browser by typing 'localhost/phpmyadmin'.
4. Click on the SQL on the top menu.
5. Enter the following sql query:- 
    create database if not exists srpss;
6. Click on Go button.
7. Now, select the srpss database from the left panel.
8. Now, click on import on the top menu.
9. In section 'File to import', click on choose file. 
10. A new dialog box will appear, go to the downloaded code repository and select the 'srpss.sql' file.
11. Click on open button.
12. Now, click on the import button at the bottom of the phpmyadmin webpage.
13. Wait a few seconds and you will see all the tables in the srpss database.
14. Similarly, create a database 'world' in phpmyadmin by following above steps.
15. And, import the 'world.sql' file following above mentioned steps.
